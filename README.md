# Sample WebRTC App

Playing with the latest spec for WebRTC.

# How to use

### Server

```
cd <project-dir>/server

npm install

node server.js
```

### Client

```
cd <project-dir>

# Python 2
python -m SimpleHTTPServer 8080

# Python 3
python -m http.server 8080

Access localhost:8080 in your browser
```
